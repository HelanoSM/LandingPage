July brings us CSS and HTML

Hello World! To wrap up July, we are proud to annouce a landing page made with CSS and HTML only (I suppose), as a task at the Udemy JS course. This my second landing page, the first one was made for the NLW, an event of the Rocktseat.

news coverage and some surprises

This landing page was quite the challenge to make since it was made without JS (to be fair, there is two lines of JS code, to close menu after the media queries). So, as one can imagine, it was quite the hurdle to make the page responsive (especialy the table). But here we are, the work it's done, if someone find any error or bug, feel free to contact me on Github. And now I say my goodbyes for now. See you next time!
